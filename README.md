# PingPong
Time-killer.

# How to play
Start the ".exe" or ".py" file (if you have install python earlier) and play.

# Players control
Movement:

<h1>Player1</h1>

  
•Move DOWN - "S"
  
•Move UP - "W"

•Restart key - "LCtrl"
  
<h1>Player2</h1>

•Move UP - "↑"
  
•Move DOWN - "↓"
  
•Restart key - "RCtrl"

# Building
For build project you need the pyinstaller library (pip install pyinstaller).

Command: **pyinstaller --onefile --noconsole --icon=path\to\icon.ico projectname.py** .

"path\to\icon.ico" - the path to your icon with format ".ico".
